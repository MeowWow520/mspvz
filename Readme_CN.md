### Simple DirectMedia Layer 2
**SDL2 是一个跨平台开发库**，旨在通过 OpenGL 和 Direct3D 提供对音频、键盘、鼠标、游戏杆和图形硬件的低级访问。它被视频游戏开发者、需要精确控制图形硬件的程序员以及需要高效处理图像和声音的研究人员所使用。SDL提供了一种简单的方法来访问硬件特性，同时仍然允许编写可移植的代码。
- **版本**
    - SDL_image: 2.8.8
    - SDL_mixer: 2.8.1
    - SDL_ttf: 2.24.0
    - SDL2: 2.32.10
